# BVSHOP × Elfsight V1 Skill Pack

這份資料包用途：

1. `SKILL.md`：完整技能規格，可放入未來 AI Skill / 專案規則。
2. `PROMPT_TEMPLATE.md`：之後要請 AI 處理 BVSHOP × Elfsight 時可直接貼上。
3. `BVSHOP_ELFSIGHT_CHECKLIST.md`：上線前檢查清單。

核心原則：

- BVSHOP 管外層版面。
- Elfsight 管 widget 本體。
- 不拆 Elfsight 原碼。
- 用行內 CSS 包成話佛牌風格。
- script 不跑時，static widget 才考慮 iframe 備援。
- floating widget 不包普通 section。
