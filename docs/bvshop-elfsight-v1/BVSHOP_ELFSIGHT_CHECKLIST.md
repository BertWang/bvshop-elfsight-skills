# BVSHOP × Elfsight V1 檢查清單

## 一、辨識
- [ ] 有沒有 `static.elfsight.com/platform/platform.js`？
- [ ] 有沒有 `elfsight-app-xxxxx`？
- [ ] 有沒有 `data-elfsight-app-lazy`？
- [ ] 有沒有 iframe Share Link？
- [ ] 是 static widget 還是 floating widget？

## 二、不可動
- [ ] 不改 script src
- [ ] 不改 elfsight-app ID
- [ ] 不刪 data-elfsight-app-lazy
- [ ] 不改 iframe src
- [ ] 不拆 Elfsight 內部 DOM

## 三、可動
- [ ] 外層 max-width
- [ ] 外層 padding / margin
- [ ] 外層 background
- [ ] border-radius
- [ ] box-shadow
- [ ] 前後標題說明
- [ ] 手機版留白

## 四、安裝方式
- [ ] Static widget 放在指定頁面區塊
- [ ] Floating widget 放在頁尾/全站碼/首頁底部
- [ ] script 不跑時提供 iframe 備援
- [ ] iframe 只用在適合的 static widget

- [ ] 不破壞 Elfsight 本體
